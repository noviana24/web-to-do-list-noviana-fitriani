# TaskFlow - To-Do List App

TaskFlow is a simple and responsive to-do list application built with PHP, MySQL, HTML, CSS, and vanilla JavaScript. It allows users to register, log in, and manage their personal tasks with priorities and completion status.

## Features

- User registration and login
- Secure password hashing with PHP
- Add, complete, and delete tasks
- Task priorities: low, medium, high
- Filter tasks by priority
- Personal task list for each user
- Responsive modern interface

## Tech Stack

- PHP 8+
- MySQL
- HTML5
- CSS3
- Vanilla JavaScript

## Requirements

Make sure you have:

- XAMPP or a similar local web server
- Apache and MySQL enabled
- PHP 8 or newer

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/noviana24/web-to-do-list-noviana-fitriani.git
   ```

2. Place the project folder in your local web server directory, such as XAMPP's htdocs folder.

3. Start Apache and MySQL from XAMPP.

4. Create a MySQL database named `todo_app`.

5. Open the app in your browser:
   ```text
   http://localhost/web%20pk%20mujib/
   ```

## Database Setup

The application will automatically create these tables if they do not exist:

- users
- tasks

## Usage

- Register a new account
- Log in with your credentials
- Add tasks with a chosen priority
- Mark tasks as completed or pending
- Delete tasks you no longer need

## Notes

This project is intended as a simple personal productivity app and is suitable for learning and basic task management.
